# Scripts Directory

This directory contains utility scripts for development and maintenance.

## Files

- `pre_upload_check.py` - Comprehensive project integrity check and upload preparation validation
  - Verifies project structure
  - Runs all tests
  - Checks code quality (lint, format)
  - Validates documentation
  - Ensures CI/CD configuration

## Usage

```bash
# Run pre-upload check
python scripts/pre_upload_check.py
```
